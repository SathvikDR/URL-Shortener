# URL Shortener

A simple URL shortener application that allows users to shorten long URLs and decode shortened URLs to their original form.

## Description

This project provides a user-friendly interface for shortening URLs and decoding shortened URLs. Users can input a long URL and generate a shortened version of it. They can also input a shortened URL to decode it back to its original form.

## Features

- **Encode URL:** Enter a long URL and generate a shortened version of it.
- **Decode URL:** Input a shortened URL and decode it to its original form.
- **User Interface:** Clean and intuitive interface for ease of use.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Clone the repository to your local machine.
2. Open `index.html` in your web browser.
3. Enter a long URL in the "Encode URL" section and click the "Shorten" button.
4. The shortened URL will be displayed below.
5. To decode a URL, enter the shortened URL in the "Decode URL" section and click the "Decode & Display" button.
6. The decoded URL will be displayed below.
